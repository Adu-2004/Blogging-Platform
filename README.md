## 🧠 What is AI-Bloggify?

**AI-Bloggify** is a next-generation blogging platform that fuses the **art of writing** with the **power of artificial intelligence**. Whether you're a seasoned writer or just starting your content journey, AI-Bloggify helps you ideate, draft, and publish blog posts — faster and smarter than ever before.

No more writer's block. No more staring at an empty editor. Just you, your ideas, and an AI co-pilot ready to turn them into polished content.

## ✨ Features at a Glance

|---|---|

## 🚀 Getting Started

### Prerequisites
Make sure you have the following installed:
- **Node.js** v18+
- **npm** or **yarn**
- **MongoDB** (local or Atlas)

---

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-bloggify.git
cd ai-bloggify
```

### 2️⃣ Setup the Backend

```bash
cd backend
npm install
```

Create a `.env` file in the `/backend` directory:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_super_secret_key
OPENAI_API_KEY=your_openai_api_key
```

Start the backend server:
```bash
npm run dev
```

### 3️⃣ Setup the Frontend

```bash
cd ../frontend
npm install
```

Create a `.env` file in the `/frontend` directory:

```env
VITE_API_URL=http://localhost:5000
```

Start the frontend:
```bash
npm run dev
```

### 4️⃣ Open in Browser

```
http://localhost:5173
```

> 🎉 You're live! Start writing your first AI-powered blog post.
